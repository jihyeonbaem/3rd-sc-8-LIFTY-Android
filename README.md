# how to run
1. Open Android Studio.
2. Opening Android Studio, click on the File menu and select "New" > "Project from Version Control"".
3. In "Git Repository URL" filed, insert next link. [https://github.com/jihyeonbaem/3rd-sc-8-LIFTY-Android
4. Click "Clone" button.
5. Ctrl + Shift + O (Sync project with Gradle files.)
6. Click "Run App" Button.
